🤖 AI助手聊天应用演示

✨ 主要功能

  • 智能对话 - 支持文字和图片混合输入
  • 现代化UI - 美观的渐变背景和毛玻璃效果
  • 响应式设计 - 完美适配桌面和移动端
  • 多会话管理 - 支持多个对话历史
  • 本地存储 - 聊天记录自动保存

🚀 快速开始

选择以下任一方式启动应用：

直接打开应用 启动本地服务器

💡 使用说明

  1. 在输入框中输入你的问题
  2. 点击图片按钮上传图片（可选）
  3. 按Enter键或点击发送按钮
  4. AI会智能分析并回复你的消息
  5. 支持多轮对话和上下文理解

📱 响应式设计

应用采用现代化的响应式设计，在不同设备上都能提供优秀的用户体验：

  • 桌面端：完整的侧边栏和主聊天区域
  • 平板端：优化的布局和触摸友好的界面
  • 手机端：紧凑的布局，隐藏侧边栏

🎨 自定义主题

你可以轻松自定义应用的主题颜色和样式：

/* 修改主题颜色 */ :root { --primary-color: #667eea; /* 主色调 */ --secondary-color: #764ba2; /* 辅助色 */ --text-color: #333; /* 文字颜色 */ --bg-color: #f8f9fa; /* 背景色 */ }
